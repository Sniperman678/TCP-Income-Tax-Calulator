Read Me
---------------------------------------------------------------------------------------
1. Make sure to compile all the relevantg .java files.

2. Run server

3. Enter port number on the prompt or press "N"

4.Run the clientside 

5. Enter port and localhost on prompt or press "N"

6. Use the following commands "Tax"

7. Enter Store command with the digits of start income, end income, basetax and tax in cents(new line for each one)

8. Enter Query command to see the stored lists

9. Enter salary to see where it fits in the window range.

10. Enter "BYE" to shut client side

11.Enter "END" to shut client and server
